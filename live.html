Couldn’t load the presentation

The browser blocked reading the .pptx file. This almost always means the page was opened straight from disk (file://), which can’t fetch local files.

Fix — serve the folder over HTTP:

• VS Code: install the Live Server extension, right-click index.html → Open with Live Server.
• Terminal: run python -m http.server 8000 in this folder, then open http://localhost:8000.
• GitHub Pages: push the repo and enable Pages — it’s served over HTTP automatically.
